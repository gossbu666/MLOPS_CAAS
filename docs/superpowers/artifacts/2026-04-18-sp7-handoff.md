## 4. Honest framing points for the final report

1. **Naive persistence beats every model at t+1** (MAE 4.46 vs best 5.12). This is expected — 24-hour air quality is dominated by short-term autocorrelation. Frame t+1 as validation-of-competence, not headline win. Alert F1 is close (0.852 vs 0.835) so the model still has operational value.

2. **XGBoost tuning overfit validation at t+7** — 100 Optuna trials drove test R² from 0.614 → 0.544. LightGBM tuning stayed honest (R² 0.600). Mention in the methodology section; this is a legitimate lesson, not a failure.

3. **Neither SP7 R² gate (> 0.65 on t+7 or t+3 LSTM) was met**, but this is likely a dataset-hardness ceiling. Persistence/seasonal-7 R² at t+7 is 0.377, so our 0.60 is adding +0.22 of real lift. The gate was aspirational, not a deliverable threshold.

4. **Statistical rigor is now in place** — every headline claim has a 95% CI. Use the bootstrap_compare output to replace any unqualified "X is better than Y" phrasings in the report.
